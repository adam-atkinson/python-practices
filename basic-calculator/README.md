# Basic Calculator 🧮

A simple Python program that demonstrates basic arithmetic operations and user input handling.

## Features ✨

- Takes two numbers as input from the user
- Performs the addition operation
- Displays the result
- Demonstrates basic Python concepts: variables, input/output, and arithmetic

## How to Run 🏃‍♂️

1. Navigate to the basic-calculator directory
2. Run the Python script:

```bash
python basic_calculator.py
```

## Usage Example 📝

Simple Calculator
Enter first number: 5
Enter second number: 3
Result: 8.0

## Learning Concepts 📚

This program demonstrates:

- Using 'input()' for user input
- Converting strings to floats with `float()`
- Basic arithmetic operations
- Printing output with `print()`
- Variable assignment and usage
