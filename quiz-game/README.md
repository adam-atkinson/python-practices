# 🧠 Quiz Game

A simple Python quiz game that tests your knowledge with multiple questions and keeps score!

## 🎮 How to Play

1. Run the `quiz_game.py` script
2. Answer each question when prompted
3. Get immediate feedback on your answers
4. See your final score at the end

## 🚀 Running the Game

Make sure you have Python installed, then run:

```bash
python quiz_game.py
```

## ✨ Features

- Multiple choice questions (math and geography)
- Score tracking system
- Case-insensitive answers for text questions
- Immediate feedback on each answer
- Final score display

## 📚 Learning Concepts

This quiz game introduces several important Python programming concepts:

- **Variables**: Using variables to store and update the score
- **User Input**: Collecting answers with the `input()` function
- **String Comparison**: Checking if answers match expected values
- **String Methods**: Using `.lower()` for case-insensitive comparison
- **Conditional Logic**: `if` statements to check answers and update scores
- **Score Tracking**: Incrementing variables with `+=` operator
- **Sequential Execution**: Running questions one after another
- **Output Formatting**: Displaying results with `print()`

## 📋 Requirements

- Python 3.x
