# 🚀 Motivational Quotes

A small Python script that grabs a random motivational phrase from a list and prints it to the terminal.

## ⭐ What it does

- Imports the `random` module.
- Defines a list called `quotes`.
- Uses `random.choice(quotes)` to pick a random quote.
- Prints the chosen quote.

## ▶️ How to run

From the folder containing the file:

```bash
python motivational_quotes.py
```

Or with Python 3 explicitly:

```bash
python3 motivational_quotes.py
```

## 🛠️ File contents

- `motivational_quotes.py` - the script that selects and prints a random quote.
- `README.md` - this documentation.

## 💡 Learning Concepts

### 1. Randomisation (`random` module)

- `random.choice(sequence)` selects a random item from a list.
- Useful for practice with non-deterministic program behavior.

### 2. Lists and data storage

- `quotes` is a Python list of strings.
- Appending, removing, or replacing elements changes program output options.

### 3. Print and output

- `print()` displays results to the console.
- Great first step toward building CLI tools.

### 4. Simple script structure

- Import dependencies at top.
- Define data.
- Perform action.
- Output result.
