# 2️⃣ Number Doubler

A simple Python program that takes a number as input and returns its doubled value.

## 📝 Description

This program demonstrates fundamental Python concepts including:

- Function definition and calls
- User input handling (string to float conversion)
- Mathematical operations
- Return statements
- Output printing

## ✨ Features

- Accept a numeric input from the user
- Define and use a custom function
- Calculate the doubled value of any number
- Display the result

## 🚀 Getting Started

### How to Run

1. Navigate to the project directory:

   ```bash
   cd number-doubler
   ```

2. Run the Python script:

   ```bash
   python number_doubler.py
   ```

3. When prompted, enter a number to double

## 💡 Example Usage

```bash
Enter a number: 5
Doubled value: 10.0
```

```bash
Enter a number: 3.5
Doubled value: 7.0
```

## 📚 Learning Concepts

This program teaches:

- **Functions**: Defining reusable functions with parameters and return values
- **Type Conversion**: Converting string input to float for calculations
- **Arithmetic Operations**: Using the multiplication operator to perform calculations
- **Function Calls**: Calling functions and using their return values
- **User Interaction**: Getting input and displaying output
