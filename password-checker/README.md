# 🔐 Password Strength Checker

A Python program that evaluates password strength based on length and character composition requirements.

## 🎮 How to Use

1. Run the `password_checker.py` script
2. Enter a password when prompted
3. Get instant feedback on password strength

## 🚀 Running the Program

Make sure you have Python installed, then run:

```bash
python password_checker.py
```

## ✨ Features

- Checks minimum password length (6 characters)
- Validates presence of letters and numbers
- Provides specific feedback for weak passwords
- Simple and fast evaluation

## 📚 Learning Concepts

This password checker demonstrates several important Python string and logic concepts:

- **User Input**: Collecting password input with `input()` function
- **String Length**: Using `len()` to check password length requirements
- **String Methods**: `isalpha()` and `isnumeric()` for character type validation
- **Conditional Logic**: `if-elif-else` chain for multiple validation checks
- **Password Security**: Basic principles of strong password requirements
- **Feedback Messages**: Providing specific error messages for different failure cases
- **Boolean Logic**: Combining conditions to determine password strength
