# 🛒 Shopping List

A simple Python program that helps you create and manage a basic shopping list.

## 📝 Description

This program demonstrates fundamental Python concepts including:

- Variables and data structures (lists)
- User input handling
- List operations (append)
- Loop iteration (for loops)
- Output printing

## ✨ Features

- Accept user input for three shopping items
- Store items in a list
- Display all items in a formatted shopping list

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your system

### How to Run

1. Navigate to the project directory:

   ```bash
   cd shopping-list
   ```

2. Run the Python script:

   ```bash
   python shopping_list.py
   ```

3. When prompted, enter three items for your shopping list

## 💡 Example Usage

```bash
Enter first item: Apples
Enter second item: Milk
Enter third item: Bread
Your Shopping List:
- Apples
- Milk
- Bread
```

## 📚 Learning Concepts

This program teaches:

- **Lists**: Creating and populating lists with the `append()` method
- **Input/Output**: Using `input()` for user interaction and `print()` for display
- **Loops**: Using `for` loops to iterate through list items
- **String formatting**: Using separators and dashes for clean output
