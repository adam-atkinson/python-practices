# 🎯 Number Guessing Game

A simple Python game where you try to guess a secret number between 1 and 10.

## 🎮 How to Play

1. Run the `guessing_game.py` script
2. Enter your guess when prompted
3. See if you got it right!

## 🚀 Running the Game

Make sure you have Python installed, then run:

```bash
python guessing_game.py
```

## ✨ Features

- Simple and fun guessing game
- Instant feedback on your guess
- Secret number is always 5 (for now!)

## 🧐 Learning Concepts

This simple game demonstrates fundamental Python programming concepts:

- **Variables**: Storing and using values (like the secret number and user guess)
- **User Input**: Getting input from the user with the `input()` function
- **Type Conversion**: Converting strings to integers using `int()`
- **Conditional Logic**: Using `if-else` statements to make decisions
- **Output**: Displaying messages to the user with `print()`
- **Basic Program Flow**: How a simple Python script executes from top to bottom

## 📋 Requirements

- Python 3.x
