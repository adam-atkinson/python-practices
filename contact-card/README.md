# Contact Card 📇

A simple Python program that creates and displays a formatted contact card with user information.

## Features ✨

- Prompts user for contact information (name, phone, email, address)
- Formats and displays information in a neat card layout
- Capitalises the name properly using title case
- Simple and interactive user interface

## How to Run 🏃‍♂️

1. Navigate to the contact-card directory
2. Run the Python script:

```bash
python contact_card.py
```

## Usage Example 📝

```bash
== Contact Card ==

Enter your name: john doe
Enter your phone number: 01234567891
Enter your email address: john.doe@email.com
Enter your address: 123 Main St, Anytown, England

--- Your Contact Card ---
Name:    John Doe
Phone:   01234567891
Email:   john.doe@email.com
Address: 123 Main St, Anytown, England
-------------------------
```

## Learning Concepts 📚

This program demonstrates:

- **User Input Handling**: Using `input()` to collect information from the user
- **String Methods**: Using `strip()` to remove whitespace and `title()` to capitalise words properly
- **String Formatting**: Using f-strings for clean and readable output formatting
- **Escape Sequences**: Using `\n` for newlines to create formatted output
- **Variable Assignment**: Storing user input in meaningful variable names
- **Output Control**: Using `print()` with formatted strings to display information in a structured way
